# Architecture Review — HTML Viewer

Thư mục này chứa các file HTML để review kiến trúc một cách trực quan.

## Files

| File | Mô tả |
|------|-------|
| [aidlc-architecture.html](aidlc-architecture.html) | Kiến trúc AIDLC 8 giai đoạn — mở trong browser để review |

## Usage

```bash
# Mở trực tiếp trong browser
xdg-open docs/html/aidlc-architecture.html

# Hoặc dùng Python server
python -m http.server 8080 -d docs/html/
```
